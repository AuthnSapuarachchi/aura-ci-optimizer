# Aura Frontend (React + Vite)

React-based frontend for the Aura CI/CD Log Analyzer.

## Tech Stack

- **React 19** - UI framework
- **Vite 7** - Build tool and dev server
- **ESLint** - Code quality

## Getting Started

### Prerequisites

- Node.js >= 18.0.0
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be live at `http://localhost:5173`

### Production Build

```bash
npm run build
```

Build output will be in the `dist/` folder.

### Preview Production Build

```bash
npm run preview
```

## Environment Configuration

The API URL is configured in `src/api.jsx`:

```javascript
export const API_URL = "https://aura-spring-backend-1.onrender.com/api/v1";
```

Update this to point to your backend server.

## Deployment

### Vercel (Recommended)

The project includes a `vercel.json` configuration file.

**Automatic Deployment:**

1. Push your code to GitHub
2. Go to [Vercel](https://vercel.com) and import your repository
3. **Important:** Set the Root Directory to `frontend` in project settings
4. Vercel will automatically detect Vite and deploy

**Project Settings in Vercel:**

- **Root Directory:** `frontend` (IMPORTANT!)
- **Framework Preset:** Vite
- **Build Command:** `npm run build` (auto-detected)
- **Output Directory:** `dist` (auto-detected)
- **Install Command:** `npm install` (auto-detected)

The `vercel.json` file handles SPA routing automatically.

### Manual Deployment

1. Build the project:

   ```bash
   npm run build
   ```

2. Deploy the `dist/` folder to any static hosting service (Vercel, Netlify, AWS S3, etc.)

## Features

- **Authentication** - User login and registration
- **Log Upload** - Manual CI/CD log file analysis
- **Dashboard** - View analysis history with status and duration
- **Real-time Updates** - Automatic refresh after new log submissions
