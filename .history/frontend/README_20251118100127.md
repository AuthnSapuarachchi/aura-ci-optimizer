# Aura Frontend (React + Vite)

React-based frontend for the Aura CI/CD Log Analyzer.

## Tech Stack

- **React 19** - UI framework
- **Vite 7** - Build tool and dev server
- **ESLint** - Code quality

## Getting Started

### Prerequisites

- Node.js >= 18.0.0
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be live at `http://localhost:5173`

### Production Build

```bash
npm run build
```

Build output will be in the `dist/` folder.

### Preview Production Build

```bash
npm run preview
```

## Environment Configuration

The API URL is configured in `src/api.jsx`:

```javascript
export const API_URL = 'https://aura-spring-backend-1.onrender.com/api/v1';
```

Update this to point to your backend server.

## Deployment

### Netlify

The project includes a `netlify.toml` configuration file in the root directory. Simply connect your repository to Netlify and it will auto-deploy.

**Build settings:**
- Build command: `npm run build`
- Publish directory: `dist`
- Base directory: `frontend`

### Manual Deployment

1. Build the project:
   ```bash
   npm run build
   ```

2. Deploy the `dist/` folder to any static hosting service (Vercel, Netlify, AWS S3, etc.)

## Features

- **Authentication** - User login and registration
- **Log Upload** - Manual CI/CD log file analysis
- **Dashboard** - View analysis history with status and duration
- **Real-time Updates** - Automatic refresh after new log submissions
